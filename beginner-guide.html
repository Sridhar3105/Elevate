🚀 Build Elevate from Scratch

A complete beginner's guide — no coding experience needed. Follow each step and you'll have a real website live on the internet!

1

Install Your Tools

Before writing any code, we need 3 free programs installed on your computer.

1

Install Node.js Required

Node.js lets your computer run JavaScript code. Think of it as the engine that powers everything.

Download the LTS version (Long Term Support) — it's the most stable.
⬇ Download Node.js (Free)

After installing, open your Terminal (Mac) or Command Prompt (Windows) and type:

node --version

You should see something like v20.11.0 — that means it worked! ✅

2

Install VS Code Required

VS Code is the program where you'll write and edit your code. It's free and made by Microsoft.

⬇ Download VS Code (Free)
After installing VS Code, also install these helpful extensions: ESLint, Prettier, and Tailwind CSS IntelliSense — search for them inside VS Code's Extensions panel.
3

Install Git Required

Git saves your code history and lets you upload your code to GitHub so Vercel can deploy it.

⬇ Download Git (Free)

After installing, verify it works:

git --version
4

How to open Terminal / Command Prompt

Windows: Press Windows key + R, type cmd, press Enter.

Mac: Press Command + Space, type Terminal, press Enter.

VS Code (easiest): Open VS Code → top menu → Terminal → New Terminal. Use this one from now on!

The terminal might look scary but you're just typing commands — like texting your computer. It does exactly what you tell it!
2

Create Your Project

One command creates the entire Next.js project structure for you automatically.

1

Choose where to save your project

First, decide where on your computer to save the project. We recommend your Desktop or Documents folder.

In your terminal, navigate there:

# Windows
cd C:\Users\YourName\Desktop

# Mac
cd ~/Desktop
cd means "change directory" — it moves you to a folder.
2

Create the Next.js project

Copy and paste this single command into your terminal and press Enter. It sets everything up automatically — takes about 2 minutes!

npx create-next-app@latest elevate --typescript --tailwind --eslint --app --src-dir --import-alias "@/*"

It will ask you some questions — just press Enter to accept the defaults for all of them.

If it asks "Would you like to use..." just press Enter (yes) for everything.
3

Open the project in VS Code

cd elevate
code .

This opens the elevate folder inside VS Code. You'll see all the project files on the left side.

4

Install all the packages

In VS Code's terminal (Terminal → New Terminal), run:

npm install lucide-react next-auth @auth/prisma-adapter prisma @prisma/client @anthropic-ai/sdk razorpay react-hook-form zod @hookform/resolvers
This downloads all the extra tools your project needs. It takes 1–2 minutes. You'll see a lot of text scrolling — that's normal!
5

Start the development server

npm run dev

Now open your browser and go to http://localhost:3000

You should see the default Next.js welcome page. 🎉 Your project is running!

To stop the server, press Ctrl + C in the terminal.
6

Your project folder structure

Here's what was created and what each folder does:

elevate/ ├── src/ │ ├── app/ ← Your pages go here │ │ ├── layout.tsx ← Sidebar, navbar wrapper │ │ ├── page.tsx ← Homepage (localhost:3000) │ │ └── api/ ← Backend code │ └── components/ ← Reusable UI pieces ├── public/ ← Images, icons ├── .env.local ← Secret keys (DO NOT share!) ├── package.json ← Project info & packages └── next.config.ts ← Next.js settings
3

Add Your Pages

Copy the Elevate app code into your project. Each page is just a file!

1

How pages work in Next.js

In Next.js, every file = one page. The file location decides the URL:

src/app/page.tsx → localhost:3000/ src/app/dashboard/page.tsx → localhost:3000/dashboard src/app/store/page.tsx → localhost:3000/store src/app/forum/page.tsx → localhost:3000/forum
Just create a folder with a page.tsx file inside and Next.js automatically creates that route!
2

Create the page folders

In your VS Code terminal, run this to create all the page folders at once:

mkdir -p src/app/dashboard src/app/profile src/app/ai src/app/courses src/app/store src/app/store/checkout src/app/library src/app/jobs src/app/forum src/app/api/ai src/app/api/auth src/components/ui src/components/layout src/lib
3

Replace the homepage

Open src/app/page.tsx in VS Code. Delete everything in it and paste this:

export default function Home() {
  return (
    <main>
      <h1>Welcome to Elevate</h1>
    </main>
  );
}
Every page file must have export default function — that's what Next.js looks for.
4

Create your Dashboard page

Create a new file: src/app/dashboard/page.tsx and paste:

export default function Dashboard() {
  return (
    <div>
      <h1>Dashboard</h1>
      <p>Welcome back, Arjun!</p>
    </div>
  );
}

Visit localhost:3000/dashboard — it's live immediately! ✅

Repeat this for every page — store, forum, ai, library, jobs, courses.

5

Create the shared Layout (Sidebar)

Open src/app/layout.tsx — this wraps every page with your sidebar and navbar.

import type { Metadata } from "next";
import "./globals.css";

export const metadata: Metadata = {
  title: "Elevate – Hardware Design Platform",
  description: "PCB designers community",
};

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en">
      <body style={{ display: "flex", background: "#07030f", color: "#f0e8ff", minHeight: "100vh" }}>
        {/* Sidebar will go here */}
        <main style={{ flex: 1 }}>
          {children}
        </main>
      </body>
    </html>
  );
}
6

Copy components from the Elevate prototype

Go back to the Elevate App artifact we built. Each component (Store, Forum, AIChat, etc.) can be copied directly into its own file:

  • Copy the Store component → paste into src/app/store/page.tsx
  • Copy the Forum component → paste into src/app/forum/page.tsx
  • Copy the AIChat component → paste into src/app/ai/page.tsx
  • Copy the Jobs component → paste into src/app/jobs/page.tsx
  • Copy the Library component → paste into src/app/library/page.tsx
  • Copy the CoursePlayer component → paste into src/app/courses/page.tsx
Add "use client"; at the very top of any page that uses useState or onClick.
4

Set Up the Database

Store users, posts, orders, and products in a free PostgreSQL database using Supabase.

1

Create a free Supabase account

Supabase gives you a free PostgreSQL database — like a giant spreadsheet in the cloud that your app reads and writes to.

🌐 Go to Supabase (Free)

Steps:

  • Click Start your project → sign in with GitHub
  • Click New Project
  • Name it elevate, choose a region close to you (e.g. Singapore for India)
  • Set a strong database password — save it somewhere!
  • Click Create new project — takes ~2 minutes to spin up
2

Get your database URL

In Supabase → Settings → Database → copy the Connection String (URI).

It looks like: postgresql://postgres:[YOUR-PASSWORD]@db.xxx.supabase.co:5432/postgres

3

Create your .env.local file

In VS Code, create a new file called .env.local in the root of your project (same level as package.json). Paste this:

# Database
DATABASE_URL="paste-your-supabase-url-here"

# NextAuth (run: npx auth secret to generate)
NEXTAUTH_URL="http://localhost:3000"
NEXTAUTH_SECRET="paste-a-random-secret-here"

# Anthropic AI
ANTHROPIC_API_KEY="sk-ant-paste-your-key-here"

# Razorpay
RAZORPAY_KEY_ID="rzp_test_paste-here"
RAZORPAY_KEY_SECRET="paste-here"
NEVER share this file or upload it to GitHub. It's already in .gitignore by default.
4

Set up Prisma (database connector)

npx prisma init

This creates a prisma/schema.prisma file. Open it and replace ALL the content with the full schema from the Next.js guide artifact.

Then run:

npx prisma generate
npx prisma db push
db push creates all your database tables automatically from the schema. You'll see tables appear in Supabase!
5

Create the Prisma client file

Create src/lib/prisma.ts and paste:

import { PrismaClient } from "@prisma/client";

const globalForPrisma = globalThis as unknown as {
  prisma: PrismaClient;
};

export const prisma =
  globalForPrisma.prisma ?? new PrismaClient();

if (process.env.NODE_ENV !== "production")
  globalForPrisma.prisma = prisma;

You get $5 free credit to start — enough for hundreds of AI conversations during development.
2

Create the AI API Route

Create file: src/app/api/ai/route.ts

import { NextRequest, NextResponse } from "next/server";
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic({
  apiKey: process.env.ANTHROPIC_API_KEY,
});

export async function POST(req: NextRequest) {
  const { messages } = await req.json();

  const response = await client.messages.create({
    model: "claude-sonnet-4-20250514",
    max_tokens: 1024,
    system: "You are Elevate AI, an expert PCB and hardware design assistant.",
    messages,
  });

  return NextResponse.json({
    reply: response.content[0]
  });
}
The API key stays on the server — users can never see it. This is safe!

Deploy & Go Live!

Put your website on the internet for free using GitHub + Vercel. Takes about 10 minutes.

1

Create a GitHub account & repo

🐙 Go to GitHub (Free)
  • Sign up at github.com
  • Click New Repository → name it elevate
  • Set it to Private (keeps your code safe)
  • Click Create Repository
2

Upload your code to GitHub

In your VS Code terminal, run these one by one:

git init
git add .
git commit -m "Initial Elevate commit"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/elevate.git
git push -u origin main
Replace YOUR-USERNAME with your actual GitHub username.
3

Deploy to Vercel (free hosting)

▲ Go to Vercel (Free)
  • Sign up at vercel.com with your GitHub account
  • Click Add New → Project
  • Find and select your elevate GitHub repo → click Import
  • Leave all settings as default
  • Click Deploy — wait ~2 minutes
Every time you push code to GitHub, Vercel automatically re-deploys. No extra steps!
4

Add your environment variables to Vercel

In Vercel → your project → Settings → Environment Variables. Add every variable from your .env.local file:

  • Add DATABASE_URL
  • Add NEXTAUTH_SECRET
  • Add NEXTAUTH_URL → change value to your Vercel domain e.g. https://elevate.vercel.app
  • Add ANTHROPIC_API_KEY
  • Add RAZORPAY_KEY_ID and RAZORPAY_KEY_SECRET

After adding all variables, click Redeploy in the Deployments tab.

5

Your site is live! 🎉

Vercel gives you a free URL like https://elevate-xyz.vercel.app. Share it with anyone!

Want a custom domain like elevate.in? Buy one from Namecheap (~₹800/yr) and connect it in Vercel → Settings → Domains.
6

Future updates — how to push changes

Every time you change your code and want to update the live site:

git add .
git commit -m "Describe what you changed"
git push

Vercel detects the push and re-deploys automatically in ~60 seconds. ⚡
